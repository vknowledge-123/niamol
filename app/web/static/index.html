Strategy

Starts ladder immediately using start preference (CALL/PUT). AUTO keeps breakout.
CALL ladder: stop = previous 1m candle low (IST). Adds only on candle close (green + delta).
PUT ladder: stop = previous 1m candle high (IST). Adds only on candle close (red + delta).
Buffer is applied to spot stop: CALL stop = low - buffer, PUT stop = high + buffer.
If OFF: first candle-close add is allowed only from the next 1m candle after entry/flip.
If ON: on TSL hit, ladder flips automatically. Overrides manual buttons.
If ON: on TSL hit, auto re-enter same ladder side. If OFF: wait for manual Flip opposite / Continue same.
If ON: after target/TSL hit, engine stops (no further trades).
If ON: square-off qty uses broker net position (slower, safer). If OFF: uses local ladder qty (faster).
